9 /**\file morse_token.h

11 
12 Each Morse code symbol is encoded as a bit vector, processed from
13 MSB to LSB with 0 corresponding to dit and 1 corresponding to
14 dah. Since Morse code symbols vary in length, a 1 bit follows the
15 last dit/dah and remaining low order bits are all zero. This makes
16 it convenient to serialize a code word by left shifting one bit at
17 a time stopping when the remainder has only the MSB set.
18 */
