1 Didah
2 =====
3 
4 Morse code by touch and tones for a personal digital assistant and amateur radio repeater voltage monitor appliance.
5 
6 Didah is a C++/Arduino application for a Teensy 3.0 microprocessor development board. The minimal hardware setup is Teensy board wired up to a pair of headphones.
7 
8 I have several goals for the project:
9 - Learn about the ARM Cortex-M4 using the Freescale Kinetis MK20DX128
10 - Develop a reusable base for home and garden automation projects
11 - See how useful a computer with two buttons for input and audio for output can be
12 - Develop a complete self-contained voltage monitor appliance for the amateur radio community
13 - Share some code with the Teensy 3.0, Arduino, and Morse Code communities
14 - Offer something to anyone who might appreciate an alternative to screens and keyboards
15 
16 The following features are present:
17 - Touch sensor auto-calibration
18 - Iambic-style Morse code entry via two touch sensors, one for dit and the other for dah.
19 - Tone generation using the pulse width modulation hardware
20 - Musical note generation by name
21 - International Morse Code prosigns and Q codes
22 - Time keeping and alarm
23 - Tutorial song ABCs as Morse code
24 - Simple arithmetic game for number practice
25 - Simple letter game for letter practice
26 - Pattern-based command recognizer
27 - Morse code USB keyboard (User taps Morse code and letters appear as if typed on a computer keyboard)
28 - Transmission rate and pitch control
29 - Analog touch sensing
30 - Report temperature
31 - Report MAC address
32 - Uptime tracking that persists across power failures
33 - Type-safety for the Morse code binary coding scheme
34 - Voltage monitoring via built-in ADC
35 - Median filtering for ADC values
36 - Calibrated temperature measurement
37 - Message format templates a la printf, but for voltage, temperature, and time
38 
39 The following features are under develpment or consideration:
40 - Low-power mode
41 - Sound effects such as different timbres for different modes
42 - Automatic input error correction
43 - List making or simple note taking
44 - Simple programming of triggers and actions via Morse code
45 - A password safe
46 - A calculator
